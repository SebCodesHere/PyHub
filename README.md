# PyHub

PyHub is a simple Python CLI toolkit that groups multiple utilities into one command.

## Tools Included

* Web Status Checker
* Calculator
* Base64 Encoder / Decoder
* Internet Speed Test
* Local Port Scanner
* Temp File Nuker
* Random Number Generator

## Run

After installation just run:

```
pyhub
```

and choose a tool from the menu.

## Install

Download the installer for the version you want and run:

```
chmod +x Install.sh
./Install.sh
```

PyHub will install to:

```
~/PyHub
```

and the `pyhub` command will work globally.

## Requirements

* Python 3
* pip
* Linux
